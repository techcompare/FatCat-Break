## 📌 Overview

**Compose GenUI** is a robust, lightweight runtime engine that parses streaming JSON payloads generated by LLMs (such as Anthropic's Claude or OpenAI's GPT-4) and dynamically compiles them into native Jetpack Compose components. 

By avoiding WebViews and raw HTML, Compose GenUI guarantees native Android performance, strict security boundaries, and seamless integration with your existing Android architecture.

## ✨ Key Features

- **Zero WebViews:** Renders 100% native Android UI via Jetpack Compose.
- **Strict Schema Enforcement:** Validates LLM hallucinations at the parsing layer before rendering.
- **Real-Time Streaming:** Designed to parse partial JSON chunks for low-latency UI generation.
- **Extensible Architecture:** Easily plug in custom Jetpack Compose components.

## 🚀 Quick Start

### 1. The Prompt
Instruct your LLM to output UI according to the Compose GenUI Schema:
```text
You are a UI generator. Output EXACTLY a valid JSON representation using the following schema.
Do not output markdown.

{
  "type": "Column" | "Row" | "Card" | "Text" | "Button",
  "children": [ ... ],
  "props": { ... }
}
```

### 2. The Implementation
Stream the response into the parser and pass the node tree to the renderer:

```kotlin
import com.genui.parser.GenUIParser
import com.genui.ui.GenerativeRenderer
import androidx.compose.runtime.Composable

@Composable
fun DashboardScreen(aiResponseJson: String) {
    // Safely parse the incoming JSON payload into the UiNode architecture
    val rootNode = GenUIParser.parse(aiResponseJson)
    
    // Dynamically render the native Jetpack Compose tree
    GenerativeRenderer(node = rootNode)
}
```
